# 🖥️ Screen Time Tracker

A modern **full-stack web and desktop-based productivity tracking system** designed to monitor, analyze, and improve digital habits through automatic screen usage tracking, real-time analytics, and cloud synchronization.

The project combines a **React.js web dashboard**, **Node.js backend APIs**, **Supabase cloud database**, and an **Electron.js desktop tracker** to provide a complete productivity management solution.

---

# 🚀 Key Features

## 🌐 Web Application
- 🔐 Secure User Authentication (JWT)
- 📊 Interactive Analytics Dashboard
- 📈 Daily & Weekly Usage Reports
- 🎯 Productivity Score Calculation
- ⏱️ Application Usage Limits
- 🌙 Light & Dark Mode Support
- ☁️ Cloud-Based Data Synchronization
- 📱 Responsive UI for Multiple Devices

---

## 🖥️ Desktop Tracker
- ⚡ Automatic Application Tracking
- 🌐 Browser Activity Detection
- 🔄 Real-Time Usage Synchronization
- 🎮 Background Tracking using Electron.js
- ▶️ Start / Stop Auto Tracking Controls
- 🪟 Windows Desktop Support
- 🔐 Secure Local Token Integration
